Step 1: Get the MySQL Pod Name
kubectl get pods


You will see something like:

mysql-746bc97748-8l4v5

Step 2: Exec into the pod
kubectl exec -it mysql-746bc97748-8l4v5 -- bash


If bash is missing (common in MySQL image), use:

kubectl exec -it mysql-746bc97748-8l4v5 -- sh

Step 3: Login to MySQL
mysql -u root -p


Password = root

To connect to your database:

      kubectl run mysql-local-client --rm --tty -i --restart='Never' --image  docker.io/bitnami/mysql:9.4.0-debian-12-r1 --namespace local --env MYSQL_ROOT_PASSWORD=$MYSQL_ROOT_PASSWORD --command -- bash

  2. To connect to primary service (read/write):

      mysql -h mysql-local.local.svc.cluster.local -uroot -p"$MYSQL_ROOT_PASSWORD"
